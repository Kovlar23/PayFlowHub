# PayFlow Hub

Учебный pet-проект: платёжная платформа уровня middle+/senior на `C#` и `.NET`.

## Status

Репозиторий инициализирован.
Следующий этап: подготовка solution structure и базового каркаса сервисов.

## Planned Stack

- `ASP.NET Core`
- `PostgreSQL`
- `EF Core`
- `Kafka`
- `RabbitMQ`
- `Redis/Valkey`
- `Camunda`
- `gRPC`
- `Docker` и `Docker Compose`
- `Kubernetes`
- `React`, `Vite`, `TypeScript`

## Workspace Notes

- Текущий SDK зафиксирован в `global.json`.
- Перед началом реализации нужно поднять Docker daemon.
- Для публикации на GitHub нужен установленный `GitHub CLI` или заранее созданный удалённый репозиторий.

Подробности по подготовке окружения: `docs/workspace-setup.md`.
